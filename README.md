# Java Mini Projects

This repository contains a collection of small Java projects created during my training as an IT specialist in application development.

## 📚 Topics Covered

* Object-Oriented Programming (OOP)
* Procedural Programming
* Basic Java syntax and logic
* Control structures (if, loops)
* Methods and functions

## 📁 Project Structure

Each folder represents a separate exercise or mini project:

Examples:

* `AllgemeinePotenz` – power calculations
* `BetragElse` – conditional logic
* `Dreieck1` – triangle calculations
* `For_1` to `For_7` – loop exercises
* `If_1` to `If_6` – conditional statements

These projects were created for learning and practicing fundamental programming concepts.
